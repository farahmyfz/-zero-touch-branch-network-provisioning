# Fase pengujian 
- 
Simulasi bencana ketika salah satu router di-reset ke pengaturan pabrik (seoah-olah rusak atau diganti baru). Praktik harus mendemonstrasikan bahwa hanya dengan satu klik (atau satu printah CI/CD), router pengganti tersebut bisa kembali beroperasi penuh seperti sedia kala dalam hitungan menit.
